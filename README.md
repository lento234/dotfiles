## Setup

1. Clone the dotfiles to your computer:

    ```bash
    $ git clone git@github.com:lento234/dotfiles.git
    $ cd dotfiles
    ```

2. Configure dotfiles:

    ```bash
    $ ./setup.sh --help
    ```
